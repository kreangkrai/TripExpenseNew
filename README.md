# Trip Expense

## คำอธิบาย
โปรแกรม Trip Expense เป็นแอปพลิเคชัน (iOS,Android) ที่ช่วยให้ผู้ใช้งานสามารถ 
สร้าง trip ในการเดินทางในแต่ละวัน เพื่อสามารถเบิกค่าน้ำมัน ค่าเบี้ยเลี้ยง ได้อย่างสะดวก และรวดเร็ว	

## ความต้องการระบบ
- Visual Studio 2022
- MAUI 9.0 C# (iOS,Android)
- MAC (Host)
